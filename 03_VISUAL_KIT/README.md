# Visual kit · Elara Nova

**Fuente canónica de assets visuales.** Lo que el sitio usa se copia a `../web/public/`.

## Carpetas

| Carpeta | Contenido |
|---------|-----------|
| `Imagenes/` | Avatar Elara, personajes objeto, fotos Evelyn B2B |
| `Logo_V1_Refinado/` | Logo sistema v1 |
| `Logo_V7_Firma/` | Firma Elara (→ `web/public/brand/`) |
| `Stickers/` | Stickers decorativos (→ `web/public/elara/stickers/`) |
| `Banners_Sociales/` | Covers FB, stories |

## Sincronización con web

Al añadir assets usados en producción, copiar a `web/public/` y actualizar `web/lib/site-images.ts` si aplica.

Duplicados byte-idénticos entre aquí y `web/public/` son **intencionales** — ver `../docs/REPO_STRUCTURE.md`.
