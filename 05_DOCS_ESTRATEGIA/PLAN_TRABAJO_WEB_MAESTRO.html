I.La idea central

Lo que vendemos es exactamente lo que estamos construyendo.

No tengo clientes B2B todavía. Tengo algo mejor: una marca propia que voy a construir con todo el detalle, toda la magia, todo el cuidado que después le voy a vender a un cliente.

soyelaranova.com es a la vez: el portal sagrado de Elara Nova, la tienda de ebooks, y el caso de éxito vivo que muestra a cualquier marca latina lo que puedo hacer por ellas.

Si una mujer entra y queda loca por el universo, compra ebook. Si un emprendedor entra y queda loco por la web, me contrata. Una sola URL, dos puertas, tres flujos de ingresos.

VII.Reglas de no-quema

Para que esto no se vuelva el agujero negro de los créditos.
Antes de cada sesión Claude Code Tener claro: qué archivo se toca, qué función se construye, cuándo se considera "hecho". Si no hay claridad, parar y planear primero (como hicimos hoy).
Qué NO codear todavía /luna (calendario menstrual). Carta natal interactiva. Login área miembros. Sanity con 40 cartas. Cohortes foro. Todo eso es 2027+. Si surge tentación, releer este párrafo.
Si surge bug grande Antes de pedirle a Claude que arregle, leer el error completo. La mayoría de bugs en R3F son de orden de import o versión de Three. 5 min de google ahorran 30 min de prompts.
El sitio ES el pitch No necesitamos brochure aparte para B2B. Cuando un prospect pide "muéstrame qué hacés", la respuesta es "abrí soyelaranova.com en pantalla completa con audio". Punto.
