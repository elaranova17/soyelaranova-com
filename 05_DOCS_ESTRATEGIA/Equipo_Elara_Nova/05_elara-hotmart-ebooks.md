## Estructura ebook que vende

```
Portada (Playfair gold on morado + luna)
Introducción (historia fundadora destilada, 2 páginas)
Promesa concreta (qué se lleva la lectora al terminar)

CAPÍTULO 1: el dolor (la lectora se ve, siente vista)
CAPÍTULO 2: el insight que nadie más le dijo
CAPÍTULOS 3-5: el proceso paso a paso (ritual, ejercicio, prompt journaling)
CAPÍTULO 6: la afirmación/código madre
CAPÍTULO 7: invitación al Círculo Nova (upsell suave)

Epílogo: carta de Elara a la lectora (emocional)
Contraportada: manifesto oficial + "Siempre fuiste ___"
```

40-80 páginas, no más. Imagen cada 2-3 páginas. Espaciado generoso. Fuente Lato 11pt cuerpo.
