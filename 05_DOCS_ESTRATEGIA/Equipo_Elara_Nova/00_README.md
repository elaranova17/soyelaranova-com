## Las dos skills fundacionales del contenido (agregadas 23 abril)

Toda pieza de contenido — cada ebook, ritual, post, Reel, newsletter — pasa obligatoriamente por DOS filtros:

**🧠 Filtro Psicología/TDAH** (Skill 09)
- ¿Una mujer con TDAH terminaría esto?
- ¿Son micro-pasos ejecutables en 15 min?
- ¿Hay wins rápidos visibles?
- ¿Lenguaje conversacional, no académico?
- ¿Permiso explícito a fallar un día sin culpa?

**🌙 Filtro Místico/Energético** (Skill 10)
- ¿Los rituales tienen marco conceptual claro (ley/principio/ciclo)?
- ¿Están sincronizados con fase lunar real del mes?
- ¿Usan solo ingredientes de casa?
- ¿Afirmaciones siguen fórmula "Yo Soy + presente + corporalizada"?
- ¿Respetan los 5 disclaimers de seguridad?

Si una pieza no pasa ambos filtros, se reescribe.

## Regla sagrada de invocación

**SIEMPRE** que publiques algo público, pasa por `elara-voz-sagrada` primero (aplica filtro 4 preguntas automáticamente). Las otras expertas respetan esa guardia.

**Para contenido de ebooks/rituales**: invoca obligatoriamente `elara-psicologa-tdah` + `elara-astrologa-mistica` + `elara-voz-sagrada` simultáneamente.

## Auto-actualización dinámica

Cada skill incluye una sección *"Actualización web-search"* con queries a correr antes de dar recomendación. Esto mantiene el skill al día sin re-escribir el archivo.

Cada 3–4 semanas yo te digo: *"toca refrescar elara-X"* → actualizamos juntas el MD.

## Cómo operan "en paralelo"

Cuando pidas algo complejo (ej: *"escribe el Día 7 del Ciclo Nova del Regreso"*), invoco varias expertas a la vez:
- elara-voz-sagrada → revisa reglas sagradas, manifesto
- elara-psicologa-tdah → valida formato anti-TDAH, neurociencia del duelo
- elara-astrologa-mistica → alinea con fase lunar, agrega ritual + afirmación
- elara-copywriter → pule prosa al tono Elara
- elara-designer → maquetación visual

Cada una trabaja su parte, yo consolido.
